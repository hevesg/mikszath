INSTALL
----------------

1. Klónozd le a repót
2. Installáld a NodeJS-t (https://www.digitalocean.com/community/tutorials/how-to-install-node-js-on-ubuntu-16-04)
3. Installáld az NPM-et (https://www.npmjs.com/get-npm)
4. Futtasd ezt a repó gyökérkönytárában: npm install
5. Installáld (adatbázis) a Drupált a szokásos módon

FEJLESZTÉS
----------------

1. Az installálás után futtasd ezt: grunt watch
